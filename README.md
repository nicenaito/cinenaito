# 映画ログアプリ

映画ログアプリは、あなたの映画視聴履歴を管理し、好みに合った映画をおすすめするWebサービスです。

## 主な機能

- **ユーザー認証**
   - ユーザー登録、ログイン、プロフィール編集が可能です。
- **映画館検索**
   - 現在地の緯度経度から、近くの映画館を検索して一覧表示します。
- **上映中映画一覧**
   - 現在上映中の映画タイトル一覧を取得できます。
- **視聴記録管理**
   - 観た映画の視聴記録を追加し、履歴を残すことができます。

## 技術スタック

- Python
- Flask
- SQLAlchemy (データベース連携)
- HTML/CSS/JavaScript (フロントエンド)

## 使い方

1. ユーザー登録を行います。
2. ログインします。 
3. 現在地の緯度経度を入力して、近くの映画館を検索します。
4. 上映中の映画一覧から観た映画を選び、視聴記録を追加します。
5. 視聴履歴を基に、おすすめの映画が提示されます。

映画の視聴記録を残すことで、あなたの好みに合ったパーソナライズされた映画おすすめを受けられます。ぜひ映画ログアプリで、次に観る映画を見つけてみてください!
