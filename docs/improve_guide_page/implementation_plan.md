# 実装計画

1. **ヒーローセクションの追加**
   - ページ上部に大きなタイトルとアプリの紹介文を配置し、歓迎ムードを演出。

2. **ステップバイステップ・ガイドの構築**
   - 1: 投稿、2: チェック、3: リアクション、の3ステップに分けて解説。
   - 各ステップに、LucideアイコンとCSS装飾を用いた「モックアップUI（擬似スクリーンショット）」を配置。
   - PCでは「説明＋画像」の横並び、スマホでは縦並びになるようグリッドレイアウトを採用。

3. **ビジュアル要素の強化**
   - `glass-card` クラスを活用し、既存のデザイン（シネマティック）と調和させる。
   - `animate-bounce` や `animate-pulse` を用いて、注目すべきポイント（投稿ボタンやクリック操作）を強調。

4. **Tipsとフッターの追加**
   - 知っていると便利な小技をまとめたカードを配置。
   - ページ末尾にダッシュボードへの大きな誘導ボタン（CTA）を配置。
