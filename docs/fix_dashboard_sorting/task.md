# タスク

ダッシュボードにおける映画予定リストの並び替え機能の修正と簡略化。

- **バグ修正**: 以前の実装ではサーバーサイドでソートしてからフィルタリングしていたため、データ件数が多い場合に選択した月のデータが取得できずリストが空になる問題があった。これをフィルタリングを先に適用するように修正する。
- **機能簡略化**: 並び替えの選択肢を「公開日順」と「投稿順」の2種類に絞り、ラベルをシンプルにする。
- **デフォルト変更**: デフォルトの並び順を「公開日順」に変更する。
- **状態保持**: 並び替え状態をURLのクエリパラメータとして保持し、リロード時にも維持されるようにする。
